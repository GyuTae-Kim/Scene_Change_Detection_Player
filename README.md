# Scene_Change_Detection_Player
### 설치 방법
- /doc 내에 있는 /ffmpeg를 C드라이브로 복사합니다
- 아래 환경 요구사항 중 tensorflow-gpu 버전을 높이기 위해서는 다음 명령을 사용합니다
- "pip3 install --upgrade --force-reinstall tensorflow-gpu==1.9.0 --user"

### 환경 요구사항
- Python 3.5 이상
- TensorFlow 1.14.0
- tensorflow-gpu 1.9.0
- Keras 2.2.4
- .Net Framework 4.6.1
